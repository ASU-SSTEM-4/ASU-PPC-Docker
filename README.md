# Dockerized ASU-PPC

## Building the docker container

Run the following:

`./docker_build.sh`

of

`sudo ./docker_build.sh`

## Initially running the docker container

Run the following:

`./docker_run.sh`

or

`sudo ./docker_run.sh`

## Getting a shell into the docker container

Run the following:

`./docker_shell.sh`

or

`sudo ./docker_shell.sh`
